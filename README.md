# AI Study Assistant

## Description
AI Study Assistant is a chatbot built using n8n and Google Gemini. It helps students by answering questions and providing educational assistance.

## Features
- AI-powered responses
- Conversational memory
- Built using n8n
- Uses Google Gemini model

## Technologies Used
- n8n
- Google Gemini
- AI Agent
- Simple Memory

## Workflow
User Message → AI Agent → Google Gemini → Response

## Future Improvements
- PDF upload support
- Flashcard generation
- Quiz generation
- RAG implementation
